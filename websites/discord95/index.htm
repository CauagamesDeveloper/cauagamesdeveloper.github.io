Discord into older operating systems.

Our plan is to bring Discord to old systems such as Windows 2000, Windows XP and Windows Vista. While this looks a bit skeptical because of security issues, we thought it would be interesting to have.

While we thought that it would be easy to make, using the Discord API and fetching the requests wasn't an easy thing, since those systems has lots of issues. For example:

- Not supporting SSL connections
- TLS 1.0/1.1 not being supported in today's standards (deprecated in 2020)

Discord95 is a project that aims Discord into older systems.
Discord95 is not affiliated with Discord Inc, neither made by the Discord team.
