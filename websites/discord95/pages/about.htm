Discord95 is a project that brings Discord to older systems

Discord95 is a project made by thecauagames to bring Discord into older operating systems, while having a easy interface and looking native.

While cauagames wanted the project to be real, he couldn't do it, sadly due to lacking programming skills.

Some months later, I decided to make the Discord95 website with new mechanics, for example, using for the first time the Bootstrap Icons, CSS transitions and tweaking the scrollbar and more!


I will use this site later for a internal development of my company, so... Better watch out!

Discord95 is a project that aims Discord into older systems.
